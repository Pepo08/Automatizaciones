# Rendiciones

Proyecto de automatización de rendición de gastos via Telegram.

## Estructura

```
rendiciones/
  workflows/
    raw/          exportación cruda desde n8n
    clean/        versión limpia para revisión y edición
    summaries/    vista rápida para análisis
  scripts/        scripts específicos de este proyecto
  docs/           documentación del proyecto
  prompts/        prompts de los agentes AI
  mappings/       mapeos de datos y transformaciones
```
